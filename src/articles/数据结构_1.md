### 算法

#### 1、数组左右移动

> 2010年

```c
// 数组 R 中的元素进行左右移动，如 abcde----(左移两位)--->cdeab
void Reverse(int R[], int from, int to){
	int i, temp;
    for ( i = 0; i < (to - from + 1) / 2; i++ ){
        temp = R[from + i];
        R[from + i] = R[to - i];
        R[to - i] = temp;
	}
}
// 数组长度 n, 左移动 p 位 
void Converse(int R[], int n, int p){
	Reverse(R, 0, p-1);
    Reverse(R, p-1, n-1);
    Reverse(R, 0, n - 1);
}
```
